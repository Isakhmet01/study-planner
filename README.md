# Study Planner

## Project Description
Study Planner is a web application for students to manage their daily tasks, subjects, and deadlines. Users can create, update, and delete tasks, organize them by subjects, and track their progress.

The project is built using:
- Frontend: Angular
- Backend: Django + Django REST Framework

## Features
- User authentication (login/logout)
- Create, read, update, delete tasks (CRUD)
- Manage subjects
- Set deadlines for tasks
- View all tasks and today's tasks
